## 🚀 快速部署步骤

### 步骤1: 准备图片（5分钟）

```bash
# 1. 创建图片目录
mkdir -p cloudflare-deploy/images

# 2. 将图片复制到目录
# 方法A: 文件管理器拖拽
# 将所有图片拖拽到 cloudflare-deploy/images/ 文件夹

# 方法B: 命令行
cp /你的图片路径/*.jpg cloudflare-deploy/images/

# 3. 验证文件
ls -lh cloudflare-deploy/images/
```

### 步骤2: 更新HTML（10分钟）

**使用查找替换功能**：

在编辑器中打开：
- `cloudflare-deploy/index.html`
- `cloudflare-deploy/products.html`

**查找**：`https://via.placeholder.com`
**替换**：`/images/`

具体替换示例：
```
查找：https://via.placeholder.com/400x300?text=Crystal+Candle+Holder
替换：/images/crystal-candle-holder.jpg
```

### 步骤3: 部署（5分钟）

1. 访问 Cloudflare Dashboard
2. 进入 Pages 项目
3. 点击 Upload assets
4. 上传 cloudflare-deploy 文件夹
5. 等待部署完成

### 步骤4: 验证（2分钟）

访问：
- https://miga.cc
- https://miga.cc/products.html

检查所有图片是否正常显示。
